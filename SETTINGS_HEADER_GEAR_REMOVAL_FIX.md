# 設定画面右上の歯車削除

## 修正した内容

設定画面を開いたときに、画面右上ヘッダーに残っていた歯車ボタン「⚙️」を確実に非表示にしました。

## 原因

設定画面内の見出し側の歯車は削除済みでしたが、ヘッダー右上の設定ボタンがCSSの強い指定により表示される可能性がありました。

## 対応内容

- 設定画面表示中に `body.settings-page-active` を付与
- 設定画面表示中は `#settingsIconButton` を強制非表示
- `hidden` / `settings-hidden` クラスでも非表示化
- `disabled` と `tabIndex=-1` を付与してキーボード操作対象からも外す

## 期待される表示

設定画面では、右上の⚙️が表示されず、上部には「← 編集画面へ戻る」のみが表示されます。
