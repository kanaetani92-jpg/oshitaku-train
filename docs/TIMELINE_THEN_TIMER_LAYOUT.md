# タイムライン→タイマー操作の表示順修正

## 目的

PC版・スマホ版ともに、タイムラインをタイマー操作より上に表示するよう修正しました。

## 表示順

### PC版

```text
1. 大きい絵カード
2. できた！／前の駅にもどる
3. タイムライン
4. タイマー操作
5. ゴール表示
```

### スマホ版

```text
1. カード
2. できた！／前の駅にもどる
3. タイムライン
4. タイマー操作
5. ゴール表示
```

## 実装

CSSで次の順番を明示しました。

- `.track-wrap`: `order: 30`
- `#viewTimerControls`: `order: 40`

また、スマホ版の `grid-template-areas` も `timeline` が `timer` より前になるよう変更しました。
