# 数字表示パネル

## 目的

設定「数字を表示」を、子どもが見る画面と編集画面の両方で見える機能として実装しました。

## 設定の意味

```text
数字を表示
残り時間と進み具合を数字で表示します
```

## 表示内容

- 残り時間
- 進み具合
- つぎまで

## 表示場所

大きい絵カードの中に表示します。

これにより、子ども画面でも編集画面でも、同じ場所で数字を確認できます。

## 表示条件

設定「数字を表示」がONのときだけ表示します。

OFFのときは、子ども画面・編集画面のどちらでも非表示になります。
