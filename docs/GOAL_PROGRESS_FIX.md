# ゴール時の進み具合修正

## 問題

ゴールに着いているのに、数字表示パネルの進み具合が `4/5` のように1つ足りなく表示されていました。

## 原因

最後の駅は「ゴール駅」として扱われ、`できた！` を押す対象ではありません。

そのため、以前の計算では次のようになっていました。

```text
完了数 = doneIndex + 1
総数 = 駅数
```

5駅の場合、最後のゴール駅に到着していても、完了区間は4つなので `4/5` と表示されていました。

## 修正

ゴール駅に到着している場合は、進み具合を満点として表示します。

```text
ゴール時：5/5
途中：完了済み数/駅数
```
