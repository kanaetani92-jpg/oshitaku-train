# タイマー専用化・前の駅にもどる機能

## 目的

「おしたくトレイン」を、当初の目的である「目に見えるタイマー」に戻すため、進み方を「できた！タイマー」へ一本化しました。

## 実装内容

- 見るモードに「前の駅にもどる」ボタンを表示
- 最初の駅では戻るボタンを無効化
- 2駅目以降では戻るボタンを有効化
- 戻ると現在駅、完了数、電車位置、タイマー位置を一つ前へ戻す
- 戻る操作の後に「元に戻す」を表示
- 自動タイマーのUIと状態遷移を削除
- 時計にあわせるUIと状態遷移を削除
- 旧保存データのauto/clock/requestedModeをtimerへ移行

## 削除した機能

- 自動タイマー
- 時計にあわせる
- 時間超過時の「自動で次へ」
- 自動タイマー終了音の設定

## 残した機能

- できた！タイマー
- 前の駅にもどる
- プレビュー
- To Do
- 設定
- PWA
- 任意ログイン準備画面
- 端末内データ層
