# 駅間時間の修正

## 問題

「おうち」駅から「きがえ」駅までの間で「できた！」を押しても、ゴールまでの残り時間が減らないことがありました。

## 原因

古い保存データでは、時間が「到着先の駅」に保存されていることがありました。

例：

```text
おうち：0分
きがえ：5分
```

現在の仕様では、時間は「この駅から次の駅まで」です。

例：

```text
おうち：5分
きがえ：5分
```

そのため、古い保存データでは「おうち→きがえ」の時間が0分として扱われ、残り時間が減りませんでした。

## 修正

古い保存データを読み込んだとき、次の条件に当てはまる場合は自動補正します。

```text
最初の駅が0分
2番目の駅が1分以上
最後の駅が0分
```

この場合、各駅の時間を1つ前へずらして、駅間時間として扱います。

## 編集画面の修正

時間欄を「この駅から次の駅までの時間」と表示し、どの区間の時間か分かるようにしました。
