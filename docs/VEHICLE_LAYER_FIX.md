# でんしゃレイヤー修正

## 問題

でんしゃが各駅アイコンの下を通るように見えていました。

## 原因

駅アイコン `.station-dot` の `z-index` が、でんしゃ `.vehicle` より高く設定されていました。

## 修正

でんしゃを駅アイコンより上のレイヤーにしました。

```css
.vehicle {
  z-index: 20 !important;
}
```

また、でんしゃの縦位置も調整し、駅アイコンの下側ではなく上側に見えるようにしました。
