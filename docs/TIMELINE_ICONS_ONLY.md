# タイムライン絵文字のみ表示

## 目的

線路上の文字が、駅アイコン・電車・線路と重なって読みにくくなる問題を避けるため、線路上から駅名を削除しました。

## 変更後の表示

線路上は絵文字だけです。

```text
🏠 ── 👕 ── 🍚 ── 🪥 ── 🎒
```

駅名は次の場所で確認します。

- 大きい絵カード
- 編集画面の駅カード

## メリット

- 文字と線路が重ならない
- 電車が見やすい
- スマホでもPCでも崩れにくい
- 子ども画面がすっきりする

## アクセシビリティ

画面上には駅名を表示しませんが、駅アイコンには `aria-label` と `title` を残しています。

## GitHub

GitHubへの反映は行っていません。
