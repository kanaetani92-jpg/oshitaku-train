# 編集画面・絵文字選択UI改善

## 目的

編集画面で、各駅の絵文字選択が分かりやすくなるように改善しました。

## 改善内容

### 駅カードの改善

各駅を1枚の編集カードとして見せます。

- 何番目の駅か
- 子ども画面でどう見えるか
- 駅名
- 絵文字
- 次の駅までの時間
- 並び替え
- 削除

を同じカード内にまとめました。

### 絵文字選択の改善

従来のように絵文字を小さな入力欄に入れるのではなく、候補ボタンから選べるようにしました。

カテゴリは次の6つです。

- よく使う
- 朝の支度
- 移動
- 学校・園
- 生活
- ごほうび

### 子ども画面プレビュー

各駅カード内に「子ども画面ではこう見えます」を表示します。

これにより、保護者や支援者が、子どもに見える画面を想像しながら編集できます。
