# ゴールまで時間表示削除

## 目的

子ども画面と編集画面から、機能としての「ゴールまで」の時間表示を削除しました。

## 削除した表示

- `remainingText`
- `timeLabel`
- `childTimeBox`
- `childRemainingText`
- 画面上の「ゴールまで」ラベル

## 残した処理

内部の時間計算は残しています。

これは、次の動作に必要なためです。

- タイマーの進行
- `できた！` を押したときの進行
- `前の駅にもどる` を押したときの戻り
- 線路上の電車位置

ただし、ゴールまでの時間は画面には表示しません。
