# Netlify用ZIP：おしたくトレイン v1.4.1

作成日：2026-06-12

このZIPはNetlifyのドラッグ＆ドロップ公開に使いやすい形式です。

## 使い方

1. Netlifyにログイン
2. Sitesで「Add new site」または「Deploy manually」を選択
3. このZIPをアップロード
4. 公開後、アプリを開く
5. 以前の版が表示される場合は、ブラウザのキャッシュまたはPWAキャッシュを更新

## 内容

- `index.html` をZIP直下に配置
- `app.js`
- `data-layer.js`
- `style.css`
- `service-worker.js`
- `manifest.webmanifest`
- `offline.html`
- `icons/`
- `netlify.toml`
- `_headers`
- `_redirects`

GitHubへのcommit、push、PR作成、ブランチ操作、Netlifyへのデプロイは行っていません。
