# おしたくトレイン タイムライン文字可読性改善版

作成日：2026-06-12

この版では、線路上の駅名が電車・丸枠・線路と重ならないように、駅名ラベルの可読性を改善しました。

## 修正内容

- 駅名を白背景の独立ラベルとして表示
- 駅名ラベルに太字・角丸・影を追加
- 駅名ラベルを線路や電車から離して配置
- 駅アイコン、電車、線路、文字の重なり順を整理
- 現在の駅と次の駅のラベルを強調
- 駅名ラベルを上下交互に配置して重なりを軽減
- 小さいスマホでは、現在の駅・次の駅以外の文字を非表示にして読みやすさを優先
- 数字表示パネル、タイムライン → タイマー操作、絵文字UI、駅間時間補正は維持
- 保存形式とPWAキャッシュをV44へ更新
