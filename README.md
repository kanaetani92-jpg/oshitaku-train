# おしたくトレイン 編集画面・絵文字選択改善版

作成日：2026-06-12

この版では、編集画面で各駅の絵文字を選びやすくし、駅カード単位で分かりやすく編集できるようにしました。

## 主な変更

- 各駅を「絵カード編集カード」として再構成
- 子ども画面での見え方を、駅カード内に大きく表示
- 絵文字を入力欄ではなく、候補ボタンから選べるように変更
- 絵文字候補をカテゴリ別に整理
  - よく使う
  - 朝の支度
  - 移動
  - 学校・園
  - 生活
  - ごほうび
- 選択中の絵文字を強調表示
- 駅名、絵文字、時間、上へ、下へ、削除を1枚のカード内に整理
- スマホでは1カラムで見やすく表示
- 保存形式とPWAキャッシュをV37へ更新

## 保存形式

`oshitakuTrainNoPhotoStateV37`

## PWAキャッシュ
