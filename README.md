# おしたくトレイン 子ども画面タイマー操作版

作成日：2026-06-12

この版では、プレビュー機能を削除し、子ども画面でタイマーの出発・一時停止を行えるようにしました。

## 変更点

- 「予定を準備する」から「プレビュー」を削除
- 「この予定で始める」を「子ども画面へ進む」に変更
- 「子ども画面へ進む」ではタイマーを開始しない
- 子ども画面に「タイマー操作」を追加
- 子ども画面で「出発・再開」「一時停止」「最初に戻る」を操作可能
- 編集画面から「4. タイマー操作」を削除
- 駅編集を「4. 駅を編集する」へ繰り上げ
- PWAキャッシュと保存形式をV30へ更新
