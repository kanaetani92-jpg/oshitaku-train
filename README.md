# おしたくトレイン 出発時の進み具合修正版

作成日：2026-06-12

この版では、数字表示パネルの「進み具合」が、出発時点で `1/5` のように表示されるように修正しました。

## 修正内容

- 出発時点の進み具合を `1/駅数` に変更
- 5駅構成なら、出発時は `1/5`
- 1つ進むと `2/5`
- ゴール時は `5/5`
- 「完了した数」ではなく「いまいる駅の位置」として進み具合を表示
- 数字表示パネル、線路上は絵文字のみ表示、タイムライン → タイマー操作、絵文字UI、駅間時間補正は維持
- 保存形式とPWAキャッシュをV47へ更新
