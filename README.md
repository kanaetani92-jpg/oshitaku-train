# おしたくトレイン 乗り物の向き修正版 1.2

作成日：2026-06-12

この版では、絵文字の船・新幹線・バスが進行方向と逆向きに見える問題を修正しました。

## 修正内容

- 船 `🚢` を進行方向に合わせて左右反転
- フェリー `⛴️`、客船 `🛳️` も同じく左右反転対象に追加
- 新幹線 `🚄`、高速鉄道 `🚅` を左右反転
- バス `🚌` を左右反転
- 反転対象以外の乗り物には影響しないように、JavaScriptで対象絵文字だけ判定
- `.vehicle-flip` クラスを追加
- でんしゃが駅アイコンの上に表示されるレイヤー修正は維持
- To Do削除、バージョン小数点表記、出発時の進み具合1始まりは維持
- バージョンを `1.2` に更新
- 保存キーを `oshitakuTrainNoPhotoState1.2` に更新
- PWAキャッシュを `oshitaku-train-pwa-1.2` に更新
