# おしたくトレイン To Do削除・バージョン1.0版

作成日：2026-06-12

この版では、To Do機能を削除し、バージョン表記を小数点表記の `1.0` に切り替えました。

## 修正内容

- To Doページを削除
- メニューの To Do 項目を削除
- To Do追加フォームを削除
- To Do一覧を削除
- To Do関連のJavaScript処理を削除
- 旧保存データに To Do が残っていても読み込み時に破棄
- 古いHTMLやPWAキャッシュが混ざった場合に備えて、CSSでもTo Do関連UIを非表示
- バージョン表記を `1.0` に変更
- 保存キーを `oshitakuTrainNoPhotoState1.0` に変更
- PWAキャッシュを `oshitaku-train-pwa-1.0` に変更
- 線路上は絵文字のみ表示、出発時の進み具合1始まり、数字表示パネルは維持
