# おしたくトレイン

子どもが「いま何をするか」「つぎに何をするか」を絵カードと路線で確認できる予定支援Webアプリです。

## 3モード化

正式名称は次の3種類です。

- **できた！タイマー**：「できた！」を押して、自分のペースで進む
- **自動タイマー**：決めた時間が過ぎると、自動的に次へ進む
- **時計にあわせる**：実際の時計に合わせて、現在の予定を表示する

基本仕様は [`docs/three-mode-spec.md`](docs/three-mode-spec.md) を参照してください。

## 現在の画面

編集画面では次の3モードを選択できます。

- できた！タイマー
- 自動タイマー
- 時計にあわせる

**できた！タイマー**は、開始、一時停止、再開、「できた！」による進行、最後のゴールまで利用できます。

**自動タイマー**は、開始、一時停止、再開、時間経過による自動進行、終了、再読込後の復元まで利用できます。

**時計にあわせる**は、開始操作なしで端末の現在時刻に対応した予定を表示します。開始前、予定中、予定の間、終了後を区別し、「できた！」やタイマー操作は表示しません。
