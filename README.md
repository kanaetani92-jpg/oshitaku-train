# おしたくトレイン 駅間時間修正版

作成日：2026-06-12

この版では、「おうち」駅から「きがえ」駅へ進んだときに残り時間が減らない問題を修正しました。

## 修正内容

- 時間を「この駅から次の駅までの時間」として明確化
- 編集画面の時間ラベルを「この駅から次の駅までの時間」に変更
- 古い保存データで `おうち.minutes = 0`、`きがえ.minutes > 0` のような場合、自動で駅間時間へ補正
- `durationBefore()` と `totalMinutes()` を駅間時間ベースで計算
- `できた！` を押したとき、最初の駅間時間も残り時間に反映
- 「前の駅にもどる」でも、戻った駅の開始時間に合わせて残り時間を戻す
- 保存形式とPWAキャッシュをV38へ更新
