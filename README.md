# おしたくトレイン 1.2ベース復帰・編集プレビュー削除版 1.2.1

作成日：2026-06-12

この版では、1つ前の `1.2` 版を基準に戻し、編集画面の「子ども画面のカード表示」系プレビューを削除しました。あわせて、`つぎまで` が `00:00` のままになる原因への安全補正を入れました。

- `1.3` の編集画面プレビュー変更を取り消し、`1.2` ベースへ戻しました

- 旧表示「子ども画面ではこう見えます」も削除
- `つぎまで` が `00:00` のままになる原因を特定
- 旧保存データで時間が「次の駅側」に保存されている場合も、現在の区間時間として扱うように補正
- `totalMinutes()` と `autoIndex()` も `intervalMinutes()` を使うようにそろえ、残り時間とつぎまでの計算モデルを統一
- 船・新幹線・バスの向き修正、でんしゃレイヤー修正、To Do削除、出発時1始まりは維持
- バージョンを `1.2.1` に更新
