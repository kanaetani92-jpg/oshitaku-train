# おしたくトレイン

子どもが「いま何をするか」「つぎに何をするか」を絵カードと路線で確認できる予定支援Webアプリです。

## 3モード化

正式名称は次の3種類です。

- **できた！タイマー**： 「できた！」を押して、自分のペースで進む
- **自動タイマー**：決めた時間が過ぎると、自動的に次へ進む
- **時計にあわせる**：実際の時計に合わせて、現在の予定を表示する

第1段階では名称・説明・進行条件・終了表現を確定しました。詳細は [`docs/three-mode-spec.md`](docs/three-mode-spec.md) を参照してください。

未実装の「自動タイマー」は、第2段階以降で保存データ、画面、時間計算、進行処理の順に追加します。
