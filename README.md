# おしたくトレイン ゴール時の進み具合修正版

作成日：2026-06-12

この版では、ゴールに着いたときに数字表示パネルの「進み具合」が `4/5` のように1つ足りなく見える問題を修正しました。

## 修正内容

- ゴール到着時は、進み具合を `駅数/駅数` と表示
- 5駅構成なら、ゴール時に `5/5` と表示
- 途中では従来どおり、完了済み数を表示
- `completedStationCount()` を追加し、進み具合の計算を明確化
- 数字表示パネル、線路上は絵文字のみ表示、タイムライン → タイマー操作、絵文字UI、駅間時間補正は維持
- 保存形式とPWAキャッシュをV46へ更新
