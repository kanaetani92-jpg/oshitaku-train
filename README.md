# おしたくトレイン タイムライン絵文字のみ表示版

作成日：2026-06-12

この版では、代替案1として、線路上は絵文字だけを表示し、駅名は出さないようにしました。

## 修正内容

- タイムライン上の駅名ラベルを非表示
- JavaScript側で線路上の駅名ラベル生成を削除
- 線路上は駅の絵文字だけを表示
- 駅名は大きい絵カード・編集画面で確認する設計に変更
- 駅名は画面上には出しませんが、アクセシビリティ用に駅アイコンへ `aria-label` と `title` を付与
- 駅名ラベル分の余白を減らし、タイムラインをコンパクト化
- 数字表示パネル、タイムライン → タイマー操作、絵文字UI、駅間時間補正は維持
- 保存形式とPWAキャッシュをV45へ更新
