# おしたくトレイン 数字表示パネル復活版

作成日：2026-06-12

この版では、設定「数字を表示」を「残り時間と進み具合を数字で表示します」として、子どもが見る画面と編集画面の両方で見えるようにしました。

## 修正内容

- 大きい絵カード内に数字表示パネルを追加
- 表示項目を3つに整理
  - 残り時間
  - 進み具合
  - つぎまで
- 設定「数字を表示」がONのときだけ表示
- 設定「数字を表示」がOFFのときは非表示
- 子ども画面・編集画面のどちらでも同じ数字表示が見えるように調整
- `できた！` や `前の駅にもどる` で数字が再描画されるように修正
- タイムライン → タイマー操作の順番、絵文字UI、駅間時間補正は維持
- 保存形式とPWAキャッシュをV43へ更新
