# 左上電車アイコン位置の統一

## 修正した問題

子ども画面と編集画面で、画面左上の電車アイコンの位置が違って見える問題を修正しました。

## 原因

スマホ幅で、編集画面では `.app` が `calc(100% - 12px)` で中央寄せになっている一方、子ども画面では `body.view-mode .app` に `width: 100%` が指定されていました。

そのため、子ども画面だけアプリ全体が画面端まで広がり、左上の電車アイコンが左端に寄って見えていました。

## 修正内容

- スマホ幅で `body.view-mode .app` と `body.edit-mode .app` の幅を統一
- 両方とも `width: calc(100% - 12px)` に統一
- 左右の `margin: auto` を明示
- 子ども画面の `.layout` で追加されていた左右余白を上書きし、画面の基準位置を編集画面と揃えました

## 期待される結果

子ども画面と編集画面で、左上の電車アイコンの位置が揃います。
